# COVID_19
Statistical analysis of COVID-19 data
## Data Files
| File Name | Description | Source |
|-----------|-------------|--------|
| UK_Projected_Population_2020.csv | Projection of the UK's population in 2020 broken down by gender and age | [ONS](https://www.ons.gov.uk/peoplepopulationandcommunity/populationandmigration/populationestimates/datasets/populationestimatesforukenglandandwalesscotlandandnorthernireland) |
| ENG_WAL_Weekly_Deaths_2020 | Weekly deaths in England and Wales from all causes, grouped by age | [ONS](https://www.ons.gov.uk/peoplepopulationandcommunity/birthsdeathsandmarriages/deaths/datasets/weeklyprovisionalfiguresondeathsregisteredinenglandandwales) |
| ENG_WAL_Weekly_COVID_Deaths.csv | Weekly deaths in England and Wales where COVID-19 is mentioned on the death certificate, grouped by age | [ONS](https://www.ons.gov.uk/peoplepopulationandcommunity/birthsdeathsandmarriages/deaths/datasets/weeklyprovisionalfiguresondeathsregisteredinenglandandwales) |
